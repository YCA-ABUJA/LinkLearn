LinkLearn Pro
Overview
LinkLearn Pro is a customizable, offline quiz and exam application built with Python and Tkinter. Designed for educational institutions like Yazz Communication Academy, it provides secure, local management of quizzes, exams, and user data via SQLite. Features include role-based dashboards, performance tracking, and export options.
Features

User Management: Admin dashboard for adding/editing teachers and students.
Quiz & Exam Creation: Teachers can create up to 4 quizzes per course and final exams with customizable questions.
Results Tracking: Independent result storage with charts and PDF/CSV exports.
Customization: Edit fields, labels, themes, and integrate a code sandbox.
Security: All data stored locally in SQLite for offline use.

Installation

Ensure Python 3.x is installed.
Install required libraries: pip install tkinter sqlite3 matplotlib (Tkinter is usually built-in).
Clone the repository: git clone <repo-url>.
Navigate to the project directory.

Usage
Run the app: python main.py

Admin Login: Manage users and settings.
Teacher Login: Create and assign quizzes/exams.
Student Login: Take quizzes and view results.
